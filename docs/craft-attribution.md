## About This Document

This document provides attribution and historical context for various CRAFT prompt framework flavors that emerged in 2024-2025. This information is for **human reference only** to ensure proper credit to prior works and authors.

**Important:** The SuperPrompt Framework uses **CRAFTER v0.2** (developed by Steff Vanhaverbeke), which is a distinct evolution with its own structure and components. This attribution document does not indicate compatibility or interchangeability with other CRAFT variants.

## SuperPrompt Framework: CRAFTER v0.2

**Author:** Steff Vanhaverbeke  
**License:** CC-BY 4.0  
**Website:** [coachsteff.live](https://coachsteff.live)

**CRAFTER v0.2 Components:**
1. **C**ontext — Situation, environment, constraints
2. **R**ole — AI expertise and perspective
3. **A**ction — Step-by-step numbered tasks
4. **F**ormat — Output structure specification
5. **T**arget & Tone — Audience profile and communication style
6. **E**xamples — Input/output demonstrations
7. **R**efining — Iteration and customization guidance

**Key Distinctions:**
- CRAFTER v0.2 is a complete framework designed for reusable, production-ready superprompts
- Includes dual-mode capability (enhancement vs. creation)
- Features comprehensive pattern library and evaluation rubric
- Emphasizes examples and refining guidance as core components
- Built for cross-tool compatibility (Claude, GPT, Gemini, Llama)
