## How to Use This Template

### For Humans (Copy-Paste-Customize)

1. **Replace all [PLACEHOLDERS]** with your specific content
2. **Delete instructional text** (everything in *italics* between asterisks)
3. **Fill in Examples** with real, complete scenarios (not placeholders)
4. **Choose a Pattern** from `/docs/patterns.md` that matches your reasoning structure
5. **Test your superprompt** with actual inputs before sharing

### For AI (Reference for Generation)

When asked to create a superprompt based on user requirements:

1. **Extract requirements** from user description (purpose, context, audience)
2. **Map to CRAFTER components** (Context, Role, Action, Format, Target & Tone, Examples, Refining)
3. **Use this template structure** but replace ALL placeholders with complete, specific content
4. **Generate 2-3 realistic examples** with full outputs (not placeholders)
5. **Select appropriate pattern** from patterns.md based on the task's reasoning needs
6. **Ensure professional quality** - production-ready, not a draft

### Quality Checklist

- [ ] Purpose is clear and specific (one sentence)
- [ ] Context describes real professional use case (not generic)
- [ ] Role defines relevant expertise (not overly broad)
- [ ] Action has 3-5 concrete steps with action verbs
- [ ] Format specifies exact output structure
- [ ] Target & Tone defines specific audience and communication style
- [ ] Examples section has 2-3 COMPLETE input/output demonstrations (no placeholders)
- [ ] Refining section addresses 4-6 common customization requests
- [ ] Pattern attribution is accurate
- [ ] No placeholder text remains in final version

## Common Use Cases for This Template

- **Content creation:** Blog posts, social media, scripts, presentations
- **Analysis & research:** Market research, competitive analysis, data synthesis

- **Creative work:** Design briefs, campaign ideation, storytelling
- **Operations:** Process documentation, workflow design, resource planning
- **Learning & development:** Training content, skill assessments, coaching prompts

**Remember:** The best superprompts are specific, tested, and production-ready. Use this template as scaffolding, not as the final product.
